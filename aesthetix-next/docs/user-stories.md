# User Stories

## MVP (Branding Site)
- **As a visitor**, I want to view a visually stunning home page so that I get an impression of the studio's premium design.
- **As a visitor**, I can read about services and see a portfolio to evaluate expertise.
- **As a visitor**, I can submit a contact/lead form to request a quote.

## Client Portal
- **As a client**, I can log in securely to access my project dashboard.
- **As a client**, I can view project status, milestones, and invoices.
- **As a client**, I can download invoices and make payments via Stripe.
- **As a client**, I can leave feedback on deliverables.

## Admin / Agency
- **As an admin**, I can create and manage client records.
- **As an admin**, I can generate invoices and track payments.
- **As an admin**, I can view analytics on leads and project progress.
